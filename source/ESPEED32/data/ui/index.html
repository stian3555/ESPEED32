About ESPEED32

ESPEED32 is a handheld slot car controller based on ESP32.

  • PWM motor control with adjustable frequency, throttle curve, anti-spin, and brake behavior.
  • Magnetic trigger support, calibration, and per-car profile storage.
  • Built-in backup/restore, firmware update (OTA), filesystem update (SPIFFS), and onboard user guide.
