About ESPEED32

ESPEED32 is a handheld slot car controller project built around ESP32 hardware.

  • PWM motor control with adjustable frequency, throttle curve, anti-spin, and brake behavior.
  • Magnetic trigger support, calibration, and per-car profile storage.
  • Built-in backup/restore, telemetry, firmware updates, storage updates, and an onboard user guide.
