Automatic Update Firmware and File System

Choose which release automatic update should install. Both firmware and filesystem will be updated together.

Downloads the selected release pair, uploads firmware first and filesystem second, then reboots once at the end.

Recommended when you want a matched official release without downloading files manually.

Firmware stage idle0%
Filesystem stage idle0%

Do not disconnect power during update.

Manual Update Firmware and File System

Select both files from the same release. The controller uploads firmware first and filesystem second, then reboots once at the end.

Use this if you already downloaded a matched firmware + SPIFFS pair.

Firmware stage idle0%
Filesystem stage idle0%

Do not disconnect power during update.

Backup Config Before Update?

Recommended: download a config backup before installing a firmware + filesystem pair, so restore is easier if settings change or something goes wrong.
